# The LLM Debates: Raising Happy Children

This project simulates a lively debate between three renowned figures on the topic of raising happy children. Each persona is powered by a different GPT model, and their responses are tailored to reflect their unique perspectives and expertise.

## How It Works

1. **Setup**: The notebook initializes the personas with their respective system prompts and assigns a GPT model to each.
2. **Debate Rounds**: Each persona responds to the others in a structured debate format, with their responses displayed in Markdown for clarity.
3. **Customization**: The models and prompts can be adjusted to explore different perspectives or topics.

## Requirements

- Python 3.8+
- OpenAI API key
- Required libraries: `openai`, `IPython`, `dotenv`

## Usage

1. Clone the repository.
2. Install the required dependencies using `pip install -r requirements.txt`.
3. Open the notebook `the-llm-debates.ipynb` and run the cells to simulate the debate.
